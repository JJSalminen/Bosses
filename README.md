# Bosses
-Käyttäjät voivat lisätä boss-vihollisia. Boss-vihollisista näkee, mistä pelistä se on ja missä se sijaitsee.

-Käyttäjä voi luoda tunnuksen ja kirjautua sovellukseen

-Käyttäjä voi lisätä, muokata ja poistaa boss-vihollisia.

-Käyttäjä voi nähdä kaikki lisätyt boss-viholliset

-Käyttäjä pystyy etsimään boss-vihollisia hakusanalla

-Käyttäjäsivu näyttää montako boss-vihollista käyttäjä on lisännyt ja listan käyttäjän lisäämistä boss-vihollisista.

-Käyttäjä pystyy valitsemaan boss-vihollisille yhden tai useamman luokituksen(esim. early game, mid game, main story/optional)

-Käyttäjät pystyvät lisäämään boss-vihollisille vinkkejä/strategioita. Boss-vihollisista näytetään vinkit/strategiat.
